## ✨ **协作三步曲 (3 Simple Rules)** ✨

忘掉复杂的流程，记住这三点就够了！

### **1. 开工前，先“同步”！🔄**

* 动手之前，记得先 **Fetch** 和 **Pull**，确保你的版本是最新鲜的，避免“穿越”作战！

### **2. 开发时，常“保存”！💾**

* 完成一个小功能就 **Commit** 一次，像玩游戏存档一样，给你的辛勤劳动一个安全的记录点。

### **3. 上传时，说“人话”！✍️**

* **Push** 之前，请在提交信息里写清楚你干了啥，比如“修复了玩家卡墙的Bug”或“添加了超酷的爆炸特效”。**拒绝使用“1”、“aaa”这种火星文！**
